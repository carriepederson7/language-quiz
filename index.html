C#

Mobile and Game Design: If you want to build apps for the Microsoft platform, you should naturally learn C#. C# is often the recommended language to use when making games through the Unity Game engine, the most popular game engine with the greatest global market share of 45% so far. The Unity community is very large and there are over 4.5 million registered developers, which means you'll easily be able to get support and also have access to tools built by fellow Unity developers.
Jobs and Earning: Reported 27,000 job postings and average $91,000 salary in 2019.

PHP

Web Development: About 80% of the top 10 million websites use PHP, including Facebook and Wikipedia. PHP is programmed with the help of built-in web development capabilities. The programmers can embed the code written in server-side programming language flawlessly into HTML code throughout the Script tag. The PHP frameworks even make it straightforward to work with web application development solutions by enabling a basic structure. Some of the examples of these frameworks are Laravel, Codeigniter, Symfony etc.
Jobs and Earning: Reported 15,000 job postings and average $89,000 salary in 2019.

JavaScript

User-Interface: JavaScript is said to be an object-oriented computer programming language that is commonly used for the purpose of creating interactive effects within the internet browsers.
Jobs and Earning: As per GitHub, it is one of the top Programming Languages 2019. Reported 36,000 job postings and average $111,000 salary in 2019.
